Library Management System
Project Overview

The AUCA Library Management System is a dynamic web application developed using Java Maven, JSP, Servlets, HTML, CSS, and JavaScript. The purpose of the system is to automate library operations by managing books, users, memberships, borrowing activities, shelves, rooms, and library locations.

The system follows the domain model provided in the UML diagram and implements role-based access control for different categories of users such as students, librarians, managers, deans, teachers, and heads of departments.

System Description

The application is designed to facilitate efficient library management by providing functionalities for:

User registration and authentication
Membership management
Book catalog management
Borrowing and returning books
Shelf and room organization
Library location management
User role management
Tracking book availability and borrowing history

The application uses:
